# LLM unlearning

- 对现有模型做“定点消除” 让它对某些内容“不再会、不会再说、不会再泄露”
  
- 满足 数据影响移除：让模型忘掉某些数据带来的记忆 ； 能力移除：让模型失去某类危险或不希望保留的能力
  
## unlearning target

遗忘任务可能呈现出多种形式，并且与遗忘目标密切相关。例如，一种可能侧重于去除数据的影响，而另一种可能强调去除模型的能力。

- 遗忘的具体数据有：
  
• 有害、有毒的 prompt-response 对

• 包含违法、不道德内容的数据

• 个人隐私信息 / 可识别信息

• 受版权保护的内容

- 遗忘的模型能力有：
  
• 某个不想保留的知识概念

• 某种危险能力

• 某类不希望模型继续生成的内容

- 遗忘的东西，既可以是具体数据样本，也可以是更抽象的知识概念、版权内容、隐私信息、有毒内容，甚至某种危险能力。从忘掉某几条具体数据
到忘掉一类内容、一个主题、或一种危险能力
